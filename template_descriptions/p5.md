# Environment: p5.js
**ID:** `p5`

## Overview
Creative coding environment using the p5.js library in Global Mode.

## Features
- **Library:** p5.js v1.9.0.
- **Mode:** Global Mode (functions `setup()` and `draw()` are defined globally).
- **Canvas:** Automatically relocated to `#root` container.

## Output UI
- **Visual:** p5.js Canvas (`<canvas>`).
- **Console:** Captures logs.

## Limitations
- **Instance Mode:** Not the default pattern.

## LLM Usage Hints
- Define `function setup() { ... }` and `function draw() { ... }`.
- Use standard p5 drawing commands (`rect`, `circle`, `background`).

## Example Code
```javascript
function setup() {
  createCanvas(400, 400);
  background(220);
}

function draw() {
  // Draw a circle that follows the mouse
  if (mouseIsPressed) {
    fill(0); // Black if pressed
  } else {
    fill(255); // White otherwise
  }
  circle(mouseX, mouseY, 30);
}
```
